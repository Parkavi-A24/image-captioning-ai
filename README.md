# Image Captioning AI

## Project Overview

Image Captioning AI is a Deep Learning project that combines Computer Vision and Natural Language Processing to generate captions for images automatically.

The application uses a pre-trained MobileNetV2 model to extract image features and generates a caption describing the image content.

## Features

* Upload image from browser
* Generate image captions
* Flask backend API
* Simple user interface
* Deep learning based image understanding

## Installation

1. Open project in VS Code

2. Install packages

pip install -r requirements.txt

3. Run backend

python app.py

4. Open frontend/index.html

5. Upload image and generate caption
